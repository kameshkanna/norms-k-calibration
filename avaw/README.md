# AVAW — Activation Vectors as Weights

This directory is a **boundary marker only**.

The AVAW project (Activation Vectors as Weights: Persistent Nudges via Bias Fusion
for Library-Free Behavioural Alignment) is a **separate research project** that lives
in its own repository.

## What belongs here

Nothing. Do not add AVAW experiment scripts, data, or results to this directory
or to the parent `norms_k_calibration_paper/` repo.

## Relationship to this repo

AVAW is cited as companion work (`\citep{avaw2026}`) in the K-calibration paper.
The two projects share:
- The observation that steering vectors can be treated as weight-space objects
- The `activation_baking/` Python library (maintained here, depended on by AVAW)

They do **not** share experiment scripts, results directories, or paper LaTeX.

## Where AVAW lives

Ask the project owner for the AVAW repo URL. It is not a subdirectory here.
